# API احراز هویت با JWT

یک سرویس API ایمن با استفاده از JSON Web Tokens (JWT) برای احراز هویت

## ویژگی‌های کلیدی

- احراز هویت کاربر با JWT
- مسیرهای محافظت شده با Middleware
- ساختار استاندارد RESTFUL


## پیش‌نیازها

- PHP 8.0 یا بالاتر
- Composer
- Docker
- سرور وب (Apache/Nginx)
- پایگاه داده (MySQL)
